📊 DATABASE DESIGN
────────────────────────────────────────────────────────────────────────────────

TRANSACTIONS TABLE (Write Model)
┌──────────────┬──────────────┬───────────────────────┐
│ id (UUID)    │ walletId     │ type (DEBIT/CREDIT)   │
│ amount       │ currency     │ status                │
│ metadata     │ createdAt    │ completedAt           │
│ relatedTx_Id │              │                       │
└──────────────┴──────────────┴───────────────────────┘

LEDGER_ENTRIES TABLE (Read Model - Append-Only)
┌──────────────┬──────────────┬───────────────────────┐
│ id (UUID)    │ walletId     │ type (DEBIT/CREDIT)   │
│ debitAmount  │ creditAmount │ balance (running)     │
│ currency     │ sourceTransId│ createdAt             │
└──────────────┴──────────────┴───────────────────────┘

IDEMPOTENCY_KEYS TABLE (Request Deduplication)
┌──────────────┬──────────────┬───────────────────────┐
│ key (PK)     │ requestHash  │ response (JSON)       │
│ statusCode   │ expiresAt    │ createdAt             │
└──────────────┴──────────────┴───────────────────────┘

2. Start with Docker Compose (Recommended):
   $ docker compose up --build
   
   Services:
   • API: http://localhost:3000/api/v1
   • Swagger Docs: http://localhost:3000/api/v1/docs
   • Database: localhost:5432

3. Development (without Docker):
   $ docker run -d \
     -e POSTGRES_DB=WalletDB \
     -e POSTGRES_USER=wallet_user \
     -e POSTGRES_PASSWORD=wallet_password \
     -p 5432:5432 \
     postgres:16-alpine
   
   $ pnpm run start:dev

4. Testing:
   $ pnpm run test              # Unit tests
   $ pnpm run test:e2e          # Integration tests
   $ pnpm run test:cov          # Coverage report

5. Code Quality:

   $ pnpm run format            # Prettier


📚 API ENDPOINTS
────────────────────────────────────────────────────────────────────────────────

WRITE MODEL (Transaction Module)
────────────────────────────────
POST   /api/v1/transactions
       Create debit/credit transaction
       Body: { type, walletId, amount, currency, metadata? }
       Idempotency-Key: optional header (24h TTL)

POST   /api/v1/transactions/transfer
       Atomic transfer between wallets
       Body: { sourceWalletId, targetWalletId, amount, currency }
       Idempotency-Key: optional header

POST   /api/v1/transactions/{id}/reversal
       Create reversal with compensation entry
       Idempotency-Key: optional header

GET    /api/v1/transactions/{id}
       Retrieve transaction details

READ MODEL (Ledger Module)
──────────────────────────
GET    /api/v1/wallets/{walletId}/balance
       Get current wallet balance
       Response: { walletId, balance, currency, lastUpdated }

GET    /api/v1/wallets/{walletId}/movements?limit=50&offset=0
       Get transaction history
       Response: [{ id, type, amount, balance, createdAt }, ...]

COMMON
──────
GET    /api/v1/health
       Health check endpoint
