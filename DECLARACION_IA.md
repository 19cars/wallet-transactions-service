# Declaración de Uso de IA

## Uso de IA

- Herramienta utilizada:
  - GitHub Copilot (modelo: Raptor mini)

- ¿Para qué la usaste?
  - Para generar la aplicación base, diseño de arquetipo y de buenas prácticas mediante Skills.

- ¿Qué código aceptaste?
  - Acepté en general el código que me propuso en mi segunda versión. Luego de ello mediante mis pruebas se fue corrigiendo y sumando funcionalidades.

- ¿Qué código descartaste?
  - Descarté el código de mi primera versión que no contaba con todas las Skills ni funcionalidades.

- ¿Qué validaste manualmente?
  - Revisé todos los endpoints, logs y tests.

- ¿Qué riesgos identificaste?
  - Funcionalidades incompletas o inconsistentes.
  - Test desactualizados con el avance del proyecto.
  - Readme.md desactualizado con el avance del proyecto.
